## Description

Falafel.fm is an online listening station curated by me (Luka.B).
The project was built with React, utilizing [Mixcloud API](https://www.mixcloud.com/developers/).

Live Demo: https://falafelfm.netlify.app/

## Tech Stack
- React
- Firestore Database
- Tachyons (CSS)
